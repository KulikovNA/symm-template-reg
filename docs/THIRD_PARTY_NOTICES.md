# Third-party notices

Юридический источник истины находится в корне:

- `THIRD_PARTY_NOTICES.md`;
- `third_party_modules.json`;
- `third_party_revisions.json`;
- `third_party_licenses/`.

Эти файлы сохранены cleanup-операцией. Активный runtime использует PyTorch,
NumPy, tqdm и Pillow; архитектурное происхождение clean-room компонентов
зафиксировано в корневых notices.
